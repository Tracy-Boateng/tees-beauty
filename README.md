## Overview

The Beauty Studio website is designed for showcasing beauty services such as lashes, makeup, and microblading. Users can view services, see past work in a gallery, book appointments online, and find the studio location via a map.

---

## Features

- **Responsive design**: Works on desktop, tablet, and mobile.  
- **Hero section**: Stylish landing with call-to-action buttons.  
- **Services section**: Displays all beauty services with modal popups for details.  
- **Gallery**: Categorized portfolio with modal lightbox.  
- **Booking form**: Submit appointment requests.  
- **Map integration**: Shows the studio location using Mapbox and Google Maps iframe.  
- **Discount section**: Highlight limited time offers.  
- **Footer**: Includes opening hours, contact info, and social links.
